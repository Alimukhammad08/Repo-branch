# РСЭМ — Профессиональный Электромонтаж

Современный корпоративный сайт для компании, специализирующейся на электромонтажных работах, проектировании и обслуживании электрических систем. Проект разработан на основе дизайна из Figma и ориентирован на презентацию услуг компании, демонстрацию экспертности и удобное взаимодействие с клиентами.

---

##  Особенности проекта

* **Полностью адаптивный интерфейс:** Сайт корректно отображается на смартфонах, планшетах и широкоформатных мониторах.
* **Компонентный подход:** Весь интерфейс разбит на независимые модули (Hero, Services, Timeline, About), что упрощает поддержку.
* **Интерактивные элементы:** Анимированные счетчики достижений (10+ лет, 200+ проектов) и стильные галереи услуг.
* **Pixel Perfect:** Точное соответствие макету Figma, включая сложную типографику и наложения (overlay) на изображениях.
* **Чистая структура:** Организованная архитектура проекта для быстрой масштабируемости.

## Технологический стек

* **Frontend:** HTML5, Tailwind
* **Дизайн:** Figma
